## How to Use This Repository

1. Navigate to the category folder you're interested in
2. Each folder contains a README.md with information about the topics covered
3. Add your solutions to the respective category folders
4. Feel free to organize solutions by language (e.g., `python/`, `java/`, `cpp/`) within each category
